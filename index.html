프로젝트 소개

엑셀로 수기 관리하던 전사 매출·프로젝트 현황을 실시간 시스템으로 전환하여, 각 본부가 이슈를 즉시 확인·관리하는 사내 전용 통합 관리 환경(웹 + 모바일 하이브리드 앱)을 구축합니다.

핵심 요구사항 분석

  • 사내 임직원 계정·조직도(본부·부서) 기반 사용자 관리 및 메뉴·기능별 권한 그룹(RBAC)
  • 사용자 접속 이력 및 주요 데이터 수정 로그 기록(감사 추적)
  • 프로젝트 마스터 등록·수정·진행상태 관리 및 코드·명칭·기간 다중 조건 검색·필터링
  • 본부별 이슈 등록·수정·히스토리 트래킹, 상태 변경 시 담당자 이메일 + 앱 푸시 알림
  • 프로젝트별 영업·매출·회계 데이터 입력 및 연·월·본부별 다차원 조회
  • 대용량 데이터 엑셀 다운로드 + 지정 양식 엑셀 업로드·유효성 검증
  • 전사/본부별 진행현황 요약 위젯, 매출 달성률·이슈 발생 현황 시각화 차트
  • 권한별 맞춤 대시보드, PC/모바일 반응형 UI, 하이브리드 앱 스토어 배포
  • 공통코드 관리, 세션 보안, DB 자동 백업 스케줄링

제안 포인트

  • 엑셀 호환을 1급 시민으로 설계 — 기존 관리 양식을 그대로 업로드/다운로드하고, 유효성 검증·에러 리포트로 안전한 데이터 이관을 보장합니다. (아마란스10 ERP 직접 연동 제외, 엑셀 기반 수동 동기화 방식 반영)
  • 단일 코드베이스 크로스플랫폼 — 웹과 하이브리드 앱을 통합 운영해 유지보수 비용을 최소화합니다. 유사 프로젝트에서 단일 코드베이스로 다중 플랫폼을 운영한 경험을 보유합니다.
  • 권한·감사 기반 보안 — RBAC + 접속/수정 로그 + 세션 관리 + DB 자동 백업으로 사내 데이터를 보호합니다. 유사 B2B 시스템에서 다계층 보안과 다중 역할 권한 체계를 구축한 경험이 있습니다.
  • 의사결정 대시보드 — 본부별 KPI 위젯과 시각화 차트로 경영진이 전사 현황을 한눈에 파악하도록 설계합니다.
  • 70~100명 동시 접속·대용량 엑셀 처리 — 비동기 처리·스트리밍·인덱싱 전략으로 성능과 안정성을 확보합니다.

PRD (Product Requirements Document)

프로젝트 요구사항을 정의하고 기능 범위를 명확히 합니다.

1. 프로젝트 배경 및 목적

엑셀 파일로 수기 관리하던 전사 매출·프로젝트 현황을 시스템화하여, 각 본부가 이슈를 실시간으로 확인·관리할 수 있는 사내 전용 웹 및 모바일 하이브리드 앱을 구축합니다.

  • 목표: 전사 프로젝트 진행 현황과 본부별 이슈를 직관적으로 파악하는 맞춤형 통합 관리 환경 구축
  • 기존 엑셀 데이터의 원활한 이관 및 PC·모바일 양쪽 접근 지원
  • 아마란스10 ERP와의 직접 API 연동은 제외, 엑셀 업/다운로드 기반 수동 동기화 방식 채택

3. 기능 요구사항

3.1 사용자 및 권한 관리

  • [필수] 사내 임직원 계정 발급 및 자체 로그인
  • [필수] 본부·부서 조직도 기반 사용자 계정 관리
  • [필수] 메뉴별·기능별 권한 그룹 설정 (RBAC)
  • [필수] 사용자 접속 이력 및 주요 데이터 수정 로그 기록

3.2 프로젝트 및 이슈 관리

  • [필수] 프로젝트 마스터 등록·수정·진행상태 관리
  • [필수] 코드·명칭·기간 등 다중 조건 검색 및 필터링
  • [필수] 본부별 이슈 등록·수정 및 히스토리 트래킹
  • [필수] 신규 이슈/상태 변경 시 담당자 이메일·앱 푸시 알림

3.3 데이터 관리 및 엑셀 연동

  • [필수] 프로젝트별 영업·매출·회계 기본 데이터 입력
  • [필수] 연·월·본부별 매출 및 프로젝트 현황 다차원 조회
  • [필수] 대용량 데이터 엑셀 다운로드
  • [필수] 지정 양식 엑셀 업로드 및 데이터 유효성 검증

3.4 대시보드 및 통계

  • [필수] 전사·본부별 프로젝트 진행 현황 요약 위젯
  • [필수] 기간별 매출 달성률·주요 이슈 시각화 차트
  • [필수] 로그인 사용자 권한별 맞춤형 대시보드

3.5 시스템 공통

  • [필수] PC 웹·모바일 반응형 UI
  • [필수] 모바일 하이브리드 앱 패키징 및 스토어 배포
  • [필수] 시스템 내 공통 코드 관리

B2B SaaS · 9주

프로젝트 개요

한국 기업의 분산된 결재·문서관리 프로세스를 단일 대시보드로 통합한 B2B SaaS 플랫폼. 7종 역할별 권한과 6계층 보안으로 업무 데이터를 보호.

핵심 기능

  • 역할 기반 권한(RBAC) 7종 · 6계층 보안(JWT, 2FA, IP 화이트리스트 등)
  • 관리자 대시보드 50+ 페이지, 120~150 API 엔드포인트, 34 데이터 엔티티
  • 토큰 기반 셀프서비스 포털, 접근 제어 및 감사 추적

본 프로젝트와의 유사점

조직도·역할 기반 권한 관리, 대규모 관리자 백오피스/대시보드, 감사 로그 등 본 PMS의 사용자·권한·대시보드 요구사항과 직접적으로 일치합니다.

기술 스택

Next.js NestJS TypeORM MySQL CASL RBAC Docker

투자조합 관리·업무 자동화 플랫폼

B2B SaaS / 핀테크 · 약 14개월

프로젝트 개요

엑셀·이메일·수작업으로 파편화된 펀드 관리·보고 업무를 단일 시스템으로 통합한 올인원 SaaS. 운용사·출자자·스타트업별 전용 대시보드로 현황을 실시간 조회.

핵심 기능

  • 50+ 페이지 대시보드, 80+ 데이터 엔티티, 200~300 API 엔드포인트
  • 다차원 데이터 조회 및 규제 보고 양식 자동 생성
  • 실시간 공동 편집, 5중 보안 아키텍처

본 프로젝트와의 유사점

엑셀 기반 수작업을 시스템으로 전환하고, 권한별 맞춤 대시보드와 연·월별 다차원 매출/현황 통계를 제공하는 구조가 본 프로젝트 목표와 동일합니다.

통합 관리 멀티플랫폼 시스템

B2B SaaS / 관리 시스템 · 약 6개월

프로젝트 개요

현장 기록·관리 업무를 모바일 앱과 웹 관리자 패널로 통합한 멀티플랫폼 시스템. 단일 코드베이스로 6개 플랫폼(Android·iOS·Web 등)을 지원하고 실시간 푸시 알림으로 이해관계자를 연결.

핵심 기능

  • 단일 코드베이스 6개 플랫폼 지원 (웹 + 모바일 하이브리드)
  • 140+ API 엔드포인트, 실시간 푸시 알림(FCM)
  • 웹 관리자 패널 + 모바일 앱 동시 운영, RBAC 6개 역할

본 프로젝트와의 유사점

PC 웹과 모바일 하이브리드 앱을 함께 제공하고, 상태 변경 시 이메일·앱 푸시 알림을 발송하는 본 프로젝트의 멀티플랫폼·알림 요구사항과 정확히 부합합니다.
