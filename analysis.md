## 3. 실현 가능성 분석 (내부용)

- **프로젝트 유형**: 풀스택 웹앱 + 모바일 하이브리드 (Feasibility: 가능, 버퍼 +15~20%)

- **AI 반영 공수(약 50% 절감)**: 약 70 M/D → 버퍼 +15% → **약 80 M/D**
- **1인 달력 환산**: 80 × (7/5) ≈ **약 112일** → 클라이언트 90일을 초과
- **판정**: 1인 기준으로는 90일 초과하나, **다수 인력 투입으로 기획/디자인 ↔ 개발 병행 시 90일 유지 가능**. 저장된 사용자 피드백 패턴(다수 인력 → 기간 유지 + 금액 85%)에 부합.
- **금액**: 90% = 40,500,000원 / 85% = 38,300,000원(45,000,000 × 0.85 = 38,250,000 → 10만원 단위 반올림). 사용자 확인 결과 **85% / 38,300,000원 / 90일** 확정.

### 클라이언트 질문 답변

**Q1. 웹과 앱(Android/iOS) 개발 경험이 있나요? 실제 서비스 중인 사례가 있다면 알려주세요.**

네, 웹과 모바일 앱(Android/iOS)을 동시에 구축·운영한 경험이 다수 있습니다. 대표적으로 현장 기록·관리 업무를 모바일 앱과 웹 관리자 패널로 통합한 멀티플랫폼 관리 시스템을 단일 코드베이스(Flutter + Next.js + NestJS)로 6개 플랫폼(Android·iOS·Web 등)에 출시하여 실제 운영 중이며, 실시간 푸시 알림(FCM)과 140개 이상의 API를 안정적으로 서비스하고 있습니다. 이 외에도 예약·CRM 기반 모바일 앱을 Android/iOS/Web 3개 플랫폼에 동시 출시한 경험이 있습니다. 본 프로젝트의 PC 웹 + 모바일 하이브리드 앱 + 스토어 배포 요구사항을 동일한 구조로 수행 가능합니다.

**Q2. ERP, 프로젝트 관리 시스템, 매출·수금 관리 시스템 구축 경험이 있나요? 있다면 구축 사례 또는 포트폴리오를 공유해 주세요.**

네, 사내 업무·관리 시스템 구축 경험이 풍부합니다. ① 기업의 분산된 결재·문서·업무 프로세스를 단일 대시보드로 통합한 B2B SaaS(7종 역할 권한, 6계층 보안, 50+ 관리자 페이지, 120~150 API)와 ② 엑셀·수작업으로 파편화된 펀드 관리·매출/현황 보고 업무를 시스템으로 전환하여 권한별 대시보드와 연·월별 다차원 통계를 제공한 올인원 SaaS(50+ 페이지, 80+ 데이터 엔티티)를 구축했습니다. 특히 ②는 "엑셀 수기 관리 → 시스템화, 다차원 매출/현황 조회"라는 본 프로젝트 목표와 동일한 문제를 해결한 사례입니다. 상세 내용은 제안서 사이트의 '유사 프로젝트 경험' 페이지와 위시켓 포트폴리오에서 확인하실 수 있습니다.

**Q3. UI/UX 디자인을 직접 진행한 경험이 있나요? 디자인 포트폴리오 또는 시안을 확인할 수 있을까요?**

네, 기획부터 UI/UX 디자인, 개발까지 직접 진행합니다. 위 프로젝트들의 화면 설계와 디자인 시스템(컴포넌트·대시보드·차트 레이아웃)을 Figma로 직접 제작했으며, 대시보드 중심의 관리 시스템 디자인 경험이 많습니다. 본 프로젝트도 화면설계서와 Figma 디자인 원본을 산출물로 제공하며, 디자인 시안은 미팅 시 포트폴리오와 함께 보여드리겠습니다.
