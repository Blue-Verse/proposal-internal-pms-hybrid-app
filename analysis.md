## 3. 실현 가능성 분석 (내부용)

- **프로젝트 유형**: 풀스택 웹앱 + 모바일 하이브리드 (Feasibility: 가능, 버퍼 +15~20%)
- **기본 공수(1인, AI 미반영 추정)**: 기획 20 + 디자인 20 + FE웹 30 + 앱 20 + BE/DB 35 + QA/배포 15 ≈ **약 140 M/D**
- **AI 반영 공수(약 50% 절감)**: 약 70 M/D → 버퍼 +15% → **약 80 M/D**
- **1인 달력 환산**: 80 × (7/5) ≈ **약 112일** → 클라이언트 90일을 초과
- **판정**: 1인 기준으로는 90일 초과하나, **다수 인력 투입으로 기획/디자인 ↔ 개발 병행 시 90일 유지 가능**. 저장된 사용자 피드백 패턴(다수 인력 → 기간 유지 + 금액 85%)에 부합.
- **금액**: 90% = 40,500,000원 / 85% = 38,300,000원(45,000,000 × 0.85 = 38,250,000 → 10만원 단위 반올림). 사용자 확인 결과 **85% / 38,300,000원 / 90일** 확정.

## 4. 포트폴리오 매칭

매칭 기준: 웹 관리자 대시보드 + SaaS B2B + 크로스플랫폼 앱 조합.

| 프로젝트 | 매칭 점수 근거 |
|---------|--------------|
| **EZ-Approve (전자결재·업무관리 SaaS)** | RBAC 7종, 6계층 보안, 대규모 관리자 백오피스/대시보드, 감사 추적 → 사용자·권한·대시보드 요구사항 직접 일치 |
| **Series-B (투자조합 관리 SaaS)** | 엑셀 수작업 → 시스템 전환, 권한별 대시보드, 연·월별 다차원 통계 → 본 프로젝트 목표와 동일 구조 |
| **Harmony Link (멀티플랫폼 관리 시스템)** | 단일 코드베이스 6개 플랫폼(웹+모바일), 140+ API, 실시간 푸시 알림 → 멀티플랫폼·알림 요구사항 부합 |

선정: EZ-Approve, Series-B, Harmony Link (3종).
